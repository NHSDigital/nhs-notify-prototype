1. Go to ServiceNow.
  2. Sign in with your NHS.net Connect account (previously NHSmail) if you have one. If you do not have an NHS.net Connect account, set up a Portal account by selecting register.
  3. Select the Submit a case tile. In the Categories menu, select Contact us. Then select the Contact us tile.
  4. In the short description field, include:
    • NHS Notify, so that your request comes to us
    • your service name
    • campaign
    • action

    For example: For NHS Notify - CSMS - normal results - translated templates provided.

  5. In the description field, include the date you need us to set your messages up by, and the following details. You can provide these details as file attachments if that's easier for you.

    Template details:
    • your template names and IDs, and the routing plans you want to add them to
    • if you're replacing a template, include the new and old template names and IDs, and the routing plan you want to add them to
    If you've not set up routing plans with us yet, tell us your:
    • template order
    • fallback times (how long we should try to send a message using one channel before trying the next)
    • whether you need to send Braille or audio CD letter templates
    • whether you need to include inserts with your letter templates
  6. For the service, select NHS Notify from the drop-down list.
  7. For the service offering, select NHS Notify templates from the drop-down list.
