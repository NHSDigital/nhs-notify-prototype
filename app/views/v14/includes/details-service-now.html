1. Sign in with your NHS.net Connect account (previously NHSmail) if you have one. If you do not have an NHS.net Connect account, set up a portal account by selecting register.
  2. In the short description field, include:
    • your service name
    • campaign
    • action

    For example: CSMS - normal results - translated templates provided.

  3. In the description field, include the date you need us to set your messages up by, and the following details. You can provide these details as file attachments if that's easier for you.

    Template details:
    • your template names and IDs, and the routing plans you want to add them to
    • if you're replacing a template, include the new and old template names and IDs, and the routing plan you want to add them to
    If you've not set up routing plans with us yet, tell us your:
    • template order
    • fallback times (how long we should try to send a message using one channel before trying the next)
    • whether you need to send Braille or audio CD letter templates
    • whether you need to include inserts with your letter templates
  4. For the service, select NHS Notify from the drop-down list.
  5. For the service offering, select NHS Notify templates from the drop-down list.
