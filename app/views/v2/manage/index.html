Create and submit a template

Use this tool to create and submit templates you want to send as messages using NHS Notify.

You can create templates for:

  • NHS app messages
  • emails
  • text messages (SMS)
  • letters

Before you start

You should only use this tool if the content of your templates has been fully approved and signed off by the appropriate stakeholders in your service or organisation.

Templates cannot be changed or edited once they have been submitted.

Any templates that you create but do not submit will not be saved. This means that you cannot save templates as drafts and edit them at a later time.
